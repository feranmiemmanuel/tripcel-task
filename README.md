## :dart: About ##

Tripcel Raily App clone

## :white_check_mark: Requirements ##

Before starting :checkered_flag:, you need to have [Git](https://git-scm.com) and [PHP 7+](https://php.net/) installed.

## :checkered_flag: Usage ##

```bash
# Installation
$ git clone https://github.com/feranmiemmanuel/tripcel-task.git
```

```bash
# fork and Clone the fork project
# Access the folder
$ cd tripcel-task

# Install dependencies
$ composer install

# Create .env and update
$ cp .env.example .env

# generate app key
$ php artisan key:generate

# serve application
$ php artisan serve
```
